Badge

Badges are used to highlight an item's status for quick recognition / notification. It can be used to display numbers, online / offline status, depending on where they are used.

Profile Badge

Profile Badge is used to show the current status of a user. You can with 3 variants : "online" or "offline" or "dnd"
